# Proyecto de Idiomas

Aplicación para el aprendizaje de idiomas.

## Estructura del proyecto

```
proyecto_de_idiomas-/
├── front/   # Aplicación cliente (interfaz de usuario)
└── back/    # Servidor / API (lógica de negocio y datos)
```

## Carpetas

- **front/** — Código del frontend (la parte visible con la que interactúa el usuario).
- **back/** — Código del backend (servidor, API y base de datos).

Instrucciones de instalación y ejecución pendientes de definir según la tecnología elegida.
